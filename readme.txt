Вкладки:
1. вайтлист
2. банлист
3. карантин
4. окна
5. импорт/экспорт
6. настройки

Управление списками: в нужной вкладке
 [+] (ENTER) добавить в список
 [-] (SHIFT+ENTER) убрать из списка

Во всех вкладках изменение списка происходит по 1,
во вкладке окон изменяются группами игроков: имена 
разделяются пробелом.

В настройках можно выключить блокнот (который слева)
и изменения стилей: расширение колонки с именами
и замена фона "предсмертной записки", а то при расширении
которой не были видны границы окна.

В FF сначала нужно поставить дополнение Greasemonkey. 
Для хрома это будет Tampermonkey.

Тестировалось только в Firefox.

Скрипт распространяется как есть. 
Никто никому ничего не обязан и никто никому ничего не гарантирует.

FAQ:
Q: Как ставить? Greasemonkey поставил.
A: Перетащи скрипт в браузер - он дальше сам подхватит.

Q: А для хрома?
A: Tampermonkey. Там создать скрипт и скопипастить содержимое.

Q: Почему так много весит?
A: Потому что ресурсы внутри в base64. Плюс React и lodash.

Q: Автокик будет?
A: Нет.

Q: Почему список банов пустой?
A: Потому что каждый составляет свой список и банит сам для себя.
